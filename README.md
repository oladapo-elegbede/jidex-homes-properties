# Jidex Homes & Properties

> Premium Property Marketplace Platform

A full-stack web application where users can browse, search, save, and inquire about properties, while agents manage listings and administrators oversee the platform.

## Tech Stack

- **Frontend:** React + Vite + Bootstrap
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **Authentication:** JWT
